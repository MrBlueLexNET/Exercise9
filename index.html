Bish-Bosh 2.0

Bish-Bosh lists all numbers between 1 and N, but some exceptions: If the number is evenly divisible with X, 'Bish' is listed instead of the number. If the number is evenly divisible by Y, 'Bosh' is listed instead the number. If the number is evenly divisible by both X and Y, "Bish-Bosh" is written instead of the number.

Input

N Loop number | X First division number | Y Second division number
